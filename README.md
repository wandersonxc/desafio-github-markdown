## 📑 Exemplos de Markdown

### Texto em Itálico e Negrito
- *Itálico* ou _itálico_  
- **Negrito** ou __negrito__  
- _**Itálico e negrito**_

### Cabeçalhos
```markdown
# H1  
## H2  
### H3  
```

### Lista Ordenada
1. Primeiro item  
2. Segundo item  
3. Terceiro item
4. Quarto item

### Lista Não Ordenada
- Item A  
- Item B  
  - Subitem A
  - Subitem B

```javascript
console.log("Olá, mundo!");
```
